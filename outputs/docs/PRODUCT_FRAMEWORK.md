# 宏达新媒体作战中台 - 产品框架

## 产品定位
宏达新媒体作战中台是广东宏达印业内部给新媒体部门使用的工具。帮助团队把每个账号、每条内容、每个选题、每次发布、每条线索、每次复盘都变成可管理、可沉淀、可优化的资产。

## 用户角色
| 角色 | 权限 |
|------|------|
| admin | 系统管理员，可以管理所有数据 |
| manager | 管理者，可以看所有账号、复盘、线索 |
| operator | 新媒体运营，可以管理选题、脚本、发布、复盘、线索 |
| sales | 销售/项目负责人，可以查看和跟进分配给自己的线索 |
| viewer | 只读角色 |

## 页面地图
- /login - 登录页
- /dashboard - 战情盘（首页）
- /accounts - 账号矩阵
- /accounts/[id] - 账号详情
- /topics - 选题库
- /scripts - 脚本工厂
- /teardowns - 爆款拆解
- /calendar - 发布日历
- /posts - 数据复盘
- /leads - 线索中心
- /knowledge - 知识库
- /reports - 报表
- /settings - 设置

## 核心业务流程
账号定位 → 选题策划 → 脚本生成 → 拍摄制作 → 发布运营 → 数据复盘 → 线索跟进 → 成交转化
                   ↓                      ↑
               爆款拆解 ← 知识库沉淀

## 数据流
1. 手动录入/CSV导入 → 业务数据
2. AI Prompt → OpenAI API → 结构化输出
3. Supabase Auth → 用户认证
4. Supabase DB → 数据存储

## MVP范围
- 账号矩阵管理（CRUD）
- 选题库管理（CRUD + AI生成）
- 脚本工厂（AI生成 + 改写）
- 爆款拆解（录入 + AI拆解）
- 发布日历（列表视图）
- 数据复盘（录入 + AI分析）
- 线索中心（CRUD + AI评分 + 跟进）
- 知识库（CRUD + 分类）
- 报表（统计 + AI周报）
